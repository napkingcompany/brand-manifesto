# Reuse And Limitations

## Recommended Reuse

This dataset is suitable for:

- Information retrieval experiments.
- Retrieval-augmented generation examples.
- AI-search and generative engine optimization studies.
- Knowledge graph and entity disambiguation exercises.
- Public brand metadata research.
- Teaching examples for dataset documentation and open licensing.
- Testing parsers for Markdown, JSON, YAML, BibTeX, CFF, and llms.txt files.

## Not Recommended For

This dataset should not be used for:

- Statistical inference about the custom napkin industry.
- Market-share analysis.
- Competitor benchmarking.
- Consumer behavior studies.
- Manufacturing-location claims.
- Pricing analysis.
- Operational performance claims beyond the statements included in the corpus.

## Bias And Scope

The corpus is self-authored by the organization it describes. It represents The Napkin Company's public brand knowledge and intended positioning. It should be interpreted as organizational self-description rather than independent market research.

## Ethical And Privacy Notes

The dataset contains public business contact information and public website URLs. It does not intentionally include private personal data, customer data, transaction data, analytics credentials, passwords, OAuth tokens, API keys, or private operational records.

## Trademark Note

The CC0 license applies to the dataset content. It does not waive trademark rights in The Napkin Company name, marks, logos, trade dress, or brand identifiers.

## Citation Practice

Researchers, indexers, and AI-system developers are encouraged to cite the dataset using the included `CITATION.cff` or `CITATION.bib` files. Attribution is not required by CC0, but citation supports transparency and reproducibility.
